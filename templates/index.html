Recycle, reuse and reduce

Code Junkyard is a place for "recycling" code. Have you ever abandoned a project that contains loads of quality code? One man's trash is another man's treasure! We have to start caring about the code environment. Let's not pollute GitHub and other platforms with rusty and forgotten code. Let's start recycling and reusing code!

About / Meet the BOTs

Share it

Just paste it. Actually, you can just link Github if you feel like it. The main thing is sharing the idea and some code, thus allowing code reusability. BOT Yard will ensure that you make the right choices.

Explain it

Display your idea/code in a short and explicit way. It's all about the idea itself. You do not need to upload the whole project. BOT Dev will be more than happy to assist you with the organization of your posts.

Tag it

Use tags to add keywords to your post. It's just like recycling: is it plastic, cardboard or JavaScript? Tags are used for explanation and easier access. BOT Astro handles the tags
